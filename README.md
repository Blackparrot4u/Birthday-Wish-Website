🎂 Amayra’s Birthday App
A fun, interactive birthday surprise built with Google AI Studio!

✨ Overview
Amayra’s Birthday App is a playful web experience designed to celebrate her special day. It combines heartfelt wishes, interactive quizzes, and a creative message board to make her birthday unforgettable.

🎮 Features
🎉 Landing Page – Cute birthday greeting with balloons, confetti, and cake animations.

📸 Photo Gallery – Memorable pictures in a slideshow format.

🕹️ Fun Zone Quiz – “Kitna jaante ho Amayra ko?” with funny results:

0 correct → “Aap Amayra ko to jaante hi nahi 😜”

1 correct → “Lagta hai aap Amayra se pyar nahi karte 😏”

2 correct → “Thoda aur effort karo 😂”

3 correct → “Wah wah! Encyclopedia of Amayra 🎉”

💌 Leave a Message – Friends can write wishes. On submit, it shows a letter‑style message:

Code
Dear Amayra 🎂💖,

[Your Message]

With love,
[Your Name]
→ Followed by: “Letter sent to Amayra 💌 — thank you for making her smile!”

🎶 Music & Vibes – Background birthday song or her favorite playlist.

🛠️ Tech Stack
Platform: Google AI Studio

Frontend: HTML, CSS, JavaScript

Animations: Confetti, balloons, playful effects

Logic: Quiz scoring + message rendering
